Frequently Asked Questions
I can't see the chat appearance changes I've made/not everything is working for me.
Refresh the stream page.
My chat doesn't look normal after the update.
Try resetting settings to defaults.
Clicking emotes in the emote menu doesn't work.
You have to have typed something in the input box first, before clicking an emote.

  • RatedEpicz's set

Announcements
Emote menu
Before clicking on an emote in the emote menu, you have to type something in the input box first.
Emote sets
If an emote set is updated, it will update 10minutes after watching a stream or when you refresh the page.
Other issues
If you have any issues where chat looks weird, is buggy or glitchy - try resetting your settings to defaults.
